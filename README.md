## Folder Structure
```
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html // Page Template
    favicon.ico
  src/
    App.css
    App.js // Root component of the application
    App.test.js
    index.css
    index.js
    logo.svg
    containers/
      PokemonList.js/ // Renders list of pokemon and display single pokemon detail card
      PokemonList.css/ // Use CSS grid layout 
    redux/
      actions/ 
      reducer/ 
      store.js
```

## To Run the application:

After clone, run following commands:

In the project directory, you can run:

### `npm install`

### `npm start`

Go to localhost:3000 to view it in the browser
